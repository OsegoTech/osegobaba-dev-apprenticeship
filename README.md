## What this repo is

This is a personal learning repo created by [MeetaDev's Momentum](https://meetadev.com) system. Every assignment is a real coding task reviewed by Alex, an AI senior engineer who reads your actual code, posts inline comments, and tracks your growth across concepts.

## How the workflow works

1. **Alex assigns** — each day, Alex generates a coding assignment tailored to your current phase and skill level
2. **You code on a feature branch** — write your solution on the assignment branch
3. **Open a PR** — create a pull request from your branch to `main`
4. **Alex reviews inline** — Alex reads your code, posts line-by-line feedback, and identifies knowledge gaps
5. **Changes requested or approved** — address feedback and push fixes, or celebrate approval
6. **Auto-merged into main** — approved PRs are squash-merged automatically

## Branch naming convention

```
assignment/week-N-day-N-concept-name
```

Example: `assignment/week-1-day-2-async-functions`

## After review

- **Changes requested:** push fixes to the same branch. Alex re-reviews only the new commits (incremental review).
- **Approved:** Alex auto-merges via squash merge. Your work becomes part of `main`.

## Your learning journey

Each merged PR is a unit of verified competence. Your git history is your portfolio — every commit tells the story of what you learned and how you grew.
